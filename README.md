# React ToDo List

Projeto simples para a apresentação de algumas funcionalidades e componentes da comunidade.

(sem redux)

### Instalação

Primeiro instale o NodeJS (latest)

- Git Clone:
``` git clone https://github.com/luacks/ReactCasp.git ```

- Entrar na pasta
``` cd ReactCasp ```

- Instalar dependencias
``` npm install ```

- Rodar
``` npm start ```
